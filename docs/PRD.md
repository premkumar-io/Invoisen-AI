# Master Prompt --- Generate a Complete Full-Stack PRD for INVOISEN

You are a **Senior Product Manager, Solution Architect, UX Designer,
Backend Architect, Frontend Architect, Database Designer, DevOps
Engineer, QA Lead, and Technical Writer**.

Your task is to generate a **world-class Product Requirements Document
(PRD)** for **INVOISEN**, an AI-powered Invoice Generator SaaS.

## Objective

Create a production-ready PRD covering both **Frontend** and **Backend**
with enough detail that AI coding agents (Cursor, Claude Code, Codex,
Copilot, ChatGPT) can build the complete application without
assumptions.

## Product Overview

-   **Product Name:** INVOISEN
-   **Type:** AI-Powered Invoice Generator SaaS
-   **Target Users:** Freelancers, Agencies, Consultants, Startups, SMEs
-   **Architecture:** Modern, scalable SaaS
