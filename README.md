# INVOISEN

> AI-Powered Invoice Generator SaaS

INVOISEN is a modern, production-ready invoicing platform designed for
freelancers, agencies, consultants, startups, and small businesses. It
combines AI-assisted invoice creation with a secure full-stack
architecture and a premium user experience.

## Features

-   AI Invoice Generator
-   Client Management
-   Invoice CRUD
-   PDF Export
-   Email Delivery (Resend)
-   Dashboard & Analytics
-   Reports
-   Multi-currency
-   GST/Tax Support
-   Role-Based Access Control
-   Responsive UI
-   Light, Dark, and Purple AI themes

## Getting Started

### Prerequisites

-   Node.js (Current Active LTS)
-   MongoDB Atlas
-   npm

### Install

``` bash
git clone https://github.com/your-username/invoisen.git
cd invoisen
```

Install frontend and backend dependencies, configure environment
variables, then start both applications.
